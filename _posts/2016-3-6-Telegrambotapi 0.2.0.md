I have released version 0.2.0 of [**telegrambotapi**](https://github.com/errebenito/telegrambotapi) after finding out that a new feature had been added to the API (but not documented in the API changelog). So it now supports silent updates. As usual, you can grab it in the releases section, or through Maven:

```
<dependency>
  <groupId>com.github.errebenito</groupId>
  <artifactId>telegrambotapi</artifactId>
  <version>0.2.0</version>
</dependency>
```

I will soon start work on a new branch to refactor some ugly parts of the code, and will publish a 1.0.0 version when that's done, that will also include any new features or API changes that may occur until then.
