It's been a while, hasn't it? About six very improductive months since the initial post, yikes! But the dry spell is over for now. I have finished a basic version of Tic Tac Toe in C++ using the awesome Qt libraries, QML for the interface and so on. 

[<img src="{{ site.baseurl }}/images/screenshot.png" alt="QuicTacToe v1.0.0" style="width: 400px;"/>](https://github.com/errebenito/QuickTacToe/)


The game, which I have given the rather obvious name of [**QuickTacToe**](https://github.com/errebenito/QuickTacToe) is playable, with all basic the features one could expect: detect who won or if it's a draw, restart a game, etc. However, currently it only supports playing against another human. Playing against the computer is a feature that will come eventually, but first I want to focus on making a proper release of [**telegrambotapi**](https://github.com/errebenito/telegrambotapi). There have been a number of changes to the API since I last wrote code for my implementation, but I have already started working on adding them, so you can expect good news soon.

And speaking of proper releases, QuickTacToe v1.0.0 can be downloaded [**here**](https://github.com/errebenito/QuickTacToe/tree/v1.0.0) (source code download, see build instructions in the README file; binaries may come later, but possibly only for Linux 64 bits at first).
