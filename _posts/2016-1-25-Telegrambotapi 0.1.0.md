As promised in the last post, I've updated [**telegrambotapi**](https://github.com/errebenito/telegrambotapi) to include all API changes up to January 20, 2016. This time I've done a [**proper release**](https://github.com/errebenito/telegrambotapi/tree/0.1.0), which you can find in the [**Releases**](https://github.com/errebenito/telegrambotapi/releases) section of the github page for the project.

I've also deployed it to Maven Central, and it should be synced to the repo within a few hours. When it is, you'll be able to use it simply by placing the following in your pom.xml:

```
<dependency>
  <groupId>com.github.errebenito</groupId>
  <artifactId>telegrambotapi</artifactId>
  <version>0.1.0</version>
</dependency>
```
